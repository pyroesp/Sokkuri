# Sokkuri
"そっくり" means "indistinguishable" or "dead ringer".  
Sokkuri will find duplicate files through hashing.

## Current Stage
- Getting file lists works
- SHA256 library has been tested and seems to work 100%
- Cleanup doesn't crash

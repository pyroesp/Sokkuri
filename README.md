# Sokkuri
Sokkuri means "indistinguishable" or "dead ringer".  
Sokkuri will find duplicate files through hashing.

## Current Stage
- Crashes at getting list of files, sometimes
- Crashes on cleanup
- SHA256 library has been tested and seems to work 100%
